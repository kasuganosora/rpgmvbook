# 数据格式

## 概述

RPG Maker MV 使用 JSON 格式存储所有游戏数据。这些文件位于 `data/` 目录中。

## 数据库文件

| 文件 | 说明 |
|------|------|
| System.json | 系统设置 |
| Actors.json | 角色数据 |
| Classes.json | 职业数据 |
| Skills.json | 技能数据 |
| Items.json | 道具数据 |
| Weapons.json | 武器数据 |
| Armors.json | 防具数据 |
| Enemies.json | 敌人数据 |
| Troops.json | 敌群数据 |
| States.json | 状态数据 |
| Animations.json | 动画数据 |
| Tilesets.json | 瓦片集 |
| CommonEvents.json | 公共事件 |
| MapInfos.json | 地图列表 |
| MapXXX.json | 地图数据 |
